Three steps of Bioinformatics
Information: images, sequences, …
Model: decision tree, neural network, …
Algorithm: number sorting algorithm, dynamic programming algorithm, …
Tip: Model is a abstract template for people to understand how to solve a problem. While algorithm is
a precise step by step plan for a computational procedure to yield an output from an input
